Agile Central: Viewport Container Template
=========================

## Overview
Stub file template that has a viewport with 5 panels

x|""""""""|""""""""""""""""""""""""""""|""""""""|x  
x| West   | North                      | East   |x  
x|        |""""""""""""""""""""""""""""|        |x  
x|        | Center                     |        |x  
x|        |""""""""""""""""""""""""""""|        |x  
x|        | South                      |        |x  
x|""""""""|""""""""""""""""""""""""""""|""""""""|x  
  
We also add listeners for:  
  
Pre-render  
Afer-render  
Add  
  
## Created with
Node.JS  
Rally App Builder  
Based on Sencha ExtJS4 (As Agile Central is built on 4.1)
